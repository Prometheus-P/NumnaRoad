# NumnaRoad Documentation

프로젝트 전체 문서

## 📚 문서 목록

### 프로젝트 개요
- **README.md** (루트) - 프로젝트 소개 및 시작 가이드
- **PLANNING.md** - 사업 기획 및 시장 분석
- **ROADMAP.md** - 개발 로드맵 및 마일스톤

### 기술 문서
- **ARCHITECTURE.md** - 시스템 아키텍처 및 자동화 플로우
- **DATABASE_SCHEMA.md** - 데이터베이스 구조 및 Collections
- **API_DOCS.md** - REST API 레퍼런스

### 운영 문서
- **DEPLOYMENT.md** - 배포 가이드 (Railway, Fly.io, VPS)
- **CONTRIBUTING.md** - 기여 가이드라인
- **CODE_OF_CONDUCT.md** - 커뮤니티 행동 강령
- **CHANGELOG.md** - 버전 히스토리

## 🎯 문서 사용 가이드

### 처음 시작하는 경우
1. README.md - 프로젝트 이해
2. PLANNING.md - 비즈니스 모델 이해
3. ARCHITECTURE.md - 기술 스택 이해
4. 각 디렉토리의 README.md - 상세 구현 가이드

### 개발자
1. ARCHITECTURE.md - 시스템 설계 파악
2. DATABASE_SCHEMA.md - DB 구조 이해
3. API_DOCS.md - API 사용법
4. CONTRIBUTING.md - 기여 방법

### 운영자
1. DEPLOYMENT.md - 배포 절차
2. ROADMAP.md - 개발 일정 확인
3. CHANGELOG.md - 버전 변경사항

## 📝 문서 작성 규칙

- 모든 문서는 Markdown 형식
- 코드 예시는 언어 명시
- 스크린샷은 `/docs/images/` 에 저장
- 외부 링크는 변경 가능성 최소화

## 🔄 문서 업데이트

문서 변경 시:
1. 해당 문서 수정
2. CHANGELOG.md에 기록
3. 버전 번호 업데이트 (필요시)
